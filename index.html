About This Darkomizer

This Darkomizer is tailored for Default Dark Mode: Expansion. It focuses on GUI textures only and does not modify text colors or shader-based elements, since the Expansion does not use .vsh shader files.

The final version will include queueing, time estimates, and live progress once the backend is connected.
